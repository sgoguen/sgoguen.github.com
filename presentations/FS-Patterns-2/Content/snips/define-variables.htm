Defining Variables

1: let aBooleanValue = true
2: let ``Can phrase be a variable name?`` = true
3: let greeting = "Hello World"
4: let answerToTheUniverse = 42
5: let pair = ("Joe", 21)
6: let (name,age) = pairF# Web Snippets

Variables are Read Only by Default

1: let fsharpIsGreat = true    //  Everything is readonly by default
2: fsharpIsGreat = false     //  This doesn't change the variableF# Web Snippets

Read/Write Variables

1: let mutable Weather = "Very Rainy"  //  Read/Write
2: Weather <- "Dry"F# Web Snippets
