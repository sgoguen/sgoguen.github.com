What is JQuery?

  • From the Website:
    jQuery is a fast, concise, JavaScript Library that simplifies how you traverse HTML documents, handle events, perform animations, and add Ajax interactions to your web pages.
  • Lightweight footprint - 19k
  • Pluggable Engine
  • Version 1.1 released yesterday!
  • More info: http://www.jquery.com/

jQuery API Overview

  • Selectors - Queries the elements in CSS or XPath syntaxes.
  • Traversing - Advanced JavaScript selector functions
  • Attributes/CSS - Changes formatting/attributes
  • Manipulation - Used for modifying elements content
  • Events - Click, DblClick, Hover, Keyboard, etc.
  • Effects - Fade In/Out, Transitions, etc.
  • AJAX - Methods for pulling content from the server
  • JavaScript Utilities - Various array and string methods

jQuery Concepts

  • Avoid writing enumeration (foreach) code. Use a convention that mimimized
  • Apply JavaScript like you apply CSS
  • Don't hog the good function names - Use anonymous functions
  • Chainable
