What is JQuery?

  • DHTML/AJAX library great for handling DHTML/AJAX problems.
  • Examples
    • $(".hideable").hide() // Hides all tags with class hidable
    • $("#my_link").click(my_click_function) // Adds click handler to tag where id="my_link"
  • More info: http://www.jquery.com/

What does unobstrusive JavaScript mean?

  • JavaScript code hooks itself up to HTML
    • No onclick="do_this()"
  • All JavaScript code can be loaded in seperate file.
  • Application works when JavaScript is turned off.
